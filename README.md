# Train ETA — Know When Your Train or Bus Will Arrive

Train ETA is a lightweight web app that shows the next arrivals for trains and buses in a clear, mobile-friendly interface. It focuses on speed, readability, and easy access to the latest ETA information so commuters can plan their journeys.

- Fast and minimal UI designed for phones and desktops
- Live ETAs and platform information where available
- Works with included datasets or public transit APIs

## Try it
- Run locally: `npm install` then `npm run dev`
- Production build: `npm run build` and serve the `dist/` folder

## Highlights
- Clean arrival cards and an at-a-glance station picker
- Lightweight React + TypeScript + Vite stack
- Designed for accessibility and quick scanning
